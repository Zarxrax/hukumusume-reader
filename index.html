Hukumusume-Reader

Introduction

This webpage is intended to present over 3500 short Japanese children's stories from the website hukumusume.com in a more friendly format for Japanese learners. This is done in two ways:

  1. Stories are sorted in order of difficulty. Each story has been analyzed by the jreadability algorithm, with the scores further weighted by story length. I have grouped the stories into three difficulty levels, and within each level the stories are fully ordered from easiest to hardest. Please understand that the algorithm isn't perfect, so the ranking can only give a rough idea of the difficulty of each story.
  2. I have created a user script for hukumusume.com to improve the readability of the site. It strips the interface and other non-essential content from around the stories, increases the font size, provides a dark mode toggle, and more. This can make the stories much more readable, especially on mobile devices.

The user script requires installing a browser extension such as Tampermonkey or Violentmonkey. This page on GreasyFork has instructions for installing the extention for different desktop and mobile browsers. Once you have installed an extension, you can find the user script here.
