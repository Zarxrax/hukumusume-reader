# hukumusume-reader
Japanese short stories organized for learners

See https://zarxrax.github.io/hukumusume-reader/ for more information.

This is a listing of over 3000 short children's stories on [hukumusume.com](http://hukumusume.com), ordered by difficulty using [jreadability](jreadability.net). This is intented to help Japanese learners more easily get into a habit of reading.

There is an accompanying user script available [here](https://greasyfork.org/en/scripts/578027-hukumusume-reader), which strips unnecessary content from around the stories, while providing a dark mode, larger font, and more. This makes the pages much more readable, particularly on mobile devices.
